# Research Plan: Karpathy Style LLM Wiki

## Target Audience
Anyone wanting to build a personal knowledge base following Andrej Karpathy's LLM Wiki pattern — developers, researchers, knowledge workers.

## Expected Deliverables
1. Context document explaining the architecture and philosophy
2. Python code for a CLI tool implementing the wiki
3. AGENTS.md file for agent-guided wiki operation

## Research Questions
1. What is the core philosophy of Karpathy's LLM Wiki?
2. What is the three-layer architecture?
3. What are the three operations (Ingest, Query, Lint)?
4. How does AGENTS.md fit into the ecosystem?
5. What Python implementations exist and how do they work?
6. What is the "idea file" concept?
