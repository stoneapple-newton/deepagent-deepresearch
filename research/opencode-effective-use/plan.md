# Research Plan: How to Use OpenCode Effectively

## Target Audience
Developers who want to use OpenCode as an AI coding agent in their terminal, IDE, or desktop. Assumes familiarity with software development but not necessarily with AI coding agents.

## Expected Report Shape
A comprehensive practical guide covering:
1. Installation and setup
2. Project initialization (AGENTS.md)
3. The Plan → Build workflow
4. Custom agents and multi-agent setups
5. MCP tools and integrations
6. Keyboard shortcuts and commands
7. Best practices and pro tips

## Time-Sensitive Claims
- OpenCode is at 160K+ GitHub stars as of mid-2026 (source: opencode.ai homepage)
- Anthropic blocked OpenCode from using Claude models in January 2026 (source: NxCode tutorial)
- Desktop app is in beta on macOS, Windows, Linux

## Sub-Questions
1. How to install, configure providers, and get started?
2. How to initialize a project with AGENTS.md effectively?
3. How does the Plan → Build cycle work?
4. How to create custom agents and multi-agent workflows?
5. How to use MCP servers, permissions, and custom tools?
6. What are the key commands, keyboard shortcuts, and power-user tips?
