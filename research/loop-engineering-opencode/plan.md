# Research Plan: Loop Engineering using OpenCode

## Topic
Loop Engineering - a new paradigm (June 2026) for working with AI coding agents by designing automated loops instead of manual prompting. OpenCode is an open-source terminal-based AI coding assistant that supports this workflow.

## Sub-Questions / Angles
1. What is Loop Engineering? (definition, origin, key figures)
2. How does OpenCode work? (architecture, features, plugin system)
3. Loop Engineering building blocks (the 5+ components of a loop)
4. Real-world loop designs using OpenCode
5. Tools comparison: OpenCode vs Claude Code vs Codex vs Cursor
6. Risks, costs, and limitations
7. Community reception and future trajectory
